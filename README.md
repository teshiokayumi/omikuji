# 今日の運勢 - おみくじアプリ

<img width="1781" height="878" alt="今日の運勢-おみくじアプリ-02-12-2026_03_25_PM" src="https://github.com/user-attachments/assets/82f7779e-4029-47e0-ba8e-0c99d327daac" />

シンプルで楽しいおみくじアプリです。ボタンをクリックすると、毎日の運勢をランダムに占えます。

## ✨ 機能

- **4つの運勢パターン**: 大吉、中吉、小吉、凶
- **結果画像表示**: 各パターンに対応した画像を表示
- **スムーズなアニメーション**: 背景色の変更と画像のスケールイン、ボタンのスライド効果
- **レスポンシブデザイン**: モバイルデバイスにも対応

## 📁 ファイル構成

```
.
├── index.html          # メインHTMLファイル
├── daikichi.jpeg       # 大吉の画像
├── cyukichi.jpeg       # 中吉の画像
├── syokichi.jpeg       # 小吉の画像
├── kyou.jpeg           # 凶の画像
└── README.md           # このファイル
```

## 🚀 使い方

1. `index.html`をWebブラウザで開きます
2. 「おみくじを引く！」ボタンをクリック
3. 結果画像と運勢が表示されます
4. 背景色が運勢に合わせて変わります

## 🎨 運勢と背景色

| 運勢 | 背景色 | 画像ファイル |
|------|--------|------------|
| 大吉 | ライトレッド | daikichi.jpeg |
| 中吉 | ライトグリーン | cyukichi.jpeg |
| 小吉 | ライトブルー | syokichi.jpeg |
| 凶 | ライトイエロー | kyou.jpeg |

## 📝 カスタマイズ

### 運勢パターンの追加・変更

`index.html`の以下の部分を編集してください：

```javascript
const patterns = [
  { label: '大吉！', color: '#ffcccc', image: 'daikichi.jpeg' },
  { label: '中吉', color: '#ccffcc', image: 'cyukichi.jpeg' },
  { label: '小吉', color: '#ccccff', image: 'syokichi.jpeg' },
  { label: '凶', color: '#ffffcc', image: 'kyou.jpeg' }
];
```

- `label`: 表示される運勢のテキスト
- `color`: 背景色（16進数カラーコード）
- `image`: 対応する画像ファイル名

### スタイルのカスタマイズ

CSSセクションで以下をカスタマイズできます：
- アニメーション速度
- ボタンのサイズと色
- 画像の最大幅
- フォントやテキストカラー

## 💻 技術スタック

- CSS3（アニメーション、トランジション）

## 📄 ライセンス

このプロジェクトは自由に使用、修正、配布できます。

## 🛠️ 開発者向け

このアプリは複雑な依存関係がなく、HTMLファイルと画像ファイルだけで動作します。
サーバーやビルドツール不要で、ローカルでもそのまま実行可能です。
